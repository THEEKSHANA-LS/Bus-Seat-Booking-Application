# 🚌 Bus Seat Booking Application

A full-stack bus seat booking system built using:

- React.js (Frontend)
- Node.js & Express.js (Backend)
- MySQL (Database)
- Tailwind CSS (UI)

## Features
- View bus routes and schedules
- Seat layout selection (2x2)
- Booking confirmation with booking ID
- Admin dashboard for managing buses and bookings

## Folder Structure
- `frontend/`– React frontend
- `backend/`– Node.js backend

## How to Run
### Backend
```bash
cd server
npm install
npm run dev
